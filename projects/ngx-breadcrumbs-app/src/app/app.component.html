Below u can see demo of ngx-breadcrumbs from this repo Ngx-breadcrumbs

Since we face the need to create more customizable breadcrumb for one of your client , we decide to bring this to next level with coderbase-ngx-breadcrumbs library. The main idea is to have a library which can cover most of case and avoid to fall in trap of design. Components of library should let developpers to bring their own design , so finally to expose data and logic needed. Library have to be small in term of size and with 0 dependency expect Angular itself . We should also create a branch for each version of Angular .
